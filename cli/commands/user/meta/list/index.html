wp user meta list

View Open Issues (0) View Closed Issues (7) Create New Issue

Options

<user>
The user login, user email, or user ID of the user to get metadata for.
[--keys=<keys>]
Limit output to metadata of specific keys.
[--fields=<fields>]
Limit the output to specific row fields. Defaults to id,meta_key,meta_value.
[--format=<format>]
Render output in a particular format.
—
default: table
options:
– table
– csv
– json
– count
– yaml
—
[--orderby=<fields>]
Set orderby which field.
—
default: id
options:
– id
– meta_key
– meta_value
—
[--order=<order>]
Set ascending or descending order.
—
default: asc
options:
– asc
– desc
—
[--unserialize]
Unserialize meta_value output.

Examples

# List user meta
$ wp user meta list 123 --keys=nickname,description,wp_capabilities
+---------+-----------------+--------------------------------+
| user_id | meta_key        | meta_value                     |
+---------+-----------------+--------------------------------+
| 123     | nickname        | supervisor                     |
| 123     | description     | Mary is a WordPress developer. |
| 123     | wp_capabilities | {"administrator":true}         |
+---------+-----------------+--------------------------------+
