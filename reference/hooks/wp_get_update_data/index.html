wp_get_update_data

apply_filters( ‘wp_get_update_data’, array $update_data, array $titles )

Filters the returned array of update data for plugins, themes, and WordPress core.

Parameters

$update_dataarray
Fetched update data.
  • counts array
    An array of counts for available plugin, theme, and WordPress updates.
  • update_title string
    Titles of available updates.
$titlesarray
An array of update counts and UI strings for available updates.
