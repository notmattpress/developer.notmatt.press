_config_wp_home()

_config_wp_home( string $url = '' ): string

This function’s access is marked private. This means it is not intended for use by plugin or theme developers, only in other core functions. It is listed here for completeness. UseWP_HOMEinstead.

Retrieves the WordPress home page URL.

Description

If the constant named ‘WP_HOME’ exists, then it will be used and returned by the function. This can be used to counter the redirection on your local development environment.

Source

function _config_wp_home( $url = '' ) {
	if ( defined( 'WP_HOME' ) ) {
		return untrailingslashit( WP_HOME );
	}
	return $url;
}
