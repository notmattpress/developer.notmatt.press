get_currentuserinfo()

get_currentuserinfo(): bool|WP_User

Populate global variables with information about the currently logged in user.

Description

See also

  • wp_get_current_user()

Return

bool|WP_User False on XMLRPC Request and invalid auth cookie, WP_User instance otherwise.
