wp_xmlrpc_server::__construct()

wp_xmlrpc_server::__construct()

Registers all of the XMLRPC methods that XMLRPC server understands.

Description

Sets up server and method property. Passes XMLRPC methods through the ‘xmlrpc_methods’ filter to allow plugins to extend or replace XML-RPC methods.
